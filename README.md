# Reconhecimento Facial com kotlin 
Esse app foi desenvolvido com a finalidade de testar e conhecer o plugin MLKit firebase.

## Principais características 
- Rápido e muito preciso.
- Não é necessário trinar o app, apenas adicionar um novo rosto.
- Salva rostos para uso futuro.
- Funciona em tempo real e offline.
- Interface fácil de usar.

## Ferramentas and Frameworks usados:
- Android Studio (Kotlin)
- CameraX
- ML Kit
- TensorFlow Lite

## Installation
Clone o repositório e abra o projeto no Android Studio.
